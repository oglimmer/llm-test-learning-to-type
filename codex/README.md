# TypeFlow Typing Trainer

A lightweight 10-finger typing practice web app with two modes:
- **Beginner**: shorter, easier sessions focused on accuracy
- **Advanced**: longer, faster sessions with harder text

It provides live performance feedback: **WPM**, **accuracy**, **errors**, and **time**.

## Run locally

From the project root:

```bash
python3 -m http.server 5173
```

Open in your browser:

```text
http://localhost:5173
```

## How to use

1. Select a mode (`Beginner` or `Advanced`).
2. Click **Start Session**.
3. Type the shown text in the input box.
4. Watch the live metrics and coaching feedback.
5. Click **Reset** to start over.
