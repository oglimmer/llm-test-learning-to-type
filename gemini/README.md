# TypeMaster - 10 Finger Typing Tutor

A modern, distraction-free web application to master the 10-finger typing technique.

## Features
- **Beginner Mode:** Structured drills for home, top, and bottom rows.
- **Advanced Mode:** Complex sentences and technical paragraphs.
- **Visual Keyboard:** Color-coded keys indicating correct finger placement.
- **Live Stats:** Real-time WPM (Words Per Minute) and Accuracy tracking.

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)

### Installation
```bash
npm install
```

### Run the App
Start the development server:
```bash
npm run dev
```
Open your browser to the URL shown in the terminal (usually `http://localhost:5173`).

### Build for Production
```bash
npm run build
```
